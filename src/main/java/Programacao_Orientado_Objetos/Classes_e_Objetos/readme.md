# Classes e Objetos.

#### Classe: O Projeto (O Molde):

> A **Classe** é apenas um desenho ou uma planta baixa. Ela não "existe" na memória do computador como algo  que você pode usar ainda. Ela apenas define:
>
 >> 1. **Atributos:** O que os objetos terão (características/dados).
 >> 2. **Métodos:** O que os objetos farão (comportamentos/ações).
>
> **Analogia:** Imagine a planta de uma casa feita por um arquiteto. Você não pode morar na planta, mas ela diz quantos quartos e banheiros a casa terá.

##

#### Objeto: A Construção (A Instância):

> O Objeto é a materialização da classe. Quando você usa a palavra new, você está "construindo" algo baseado naquele molde. Cada objeto é independente.
 >> **Analogia:** Usando a mesma planta (Classe), você pode construir 10 casas (Objetos) na mesma rua. Uma casa pode ser azul, a outra amarela, mas todas seguem o mesmo desenho.
> 

##

#### Resumo:

1. **Classe:** É o substantivo abstrato (ex: Pessoa, Carro, ContaBancaria).
2. **Objeto:** É o substantivo concreto (ex: João, MeuFusca, MinhaConta).
3. **Instanciar:** É o ato de criar um objeto a partir de uma classe (usando o new).
